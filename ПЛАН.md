### Этап 3: Создать Chatterbox сервис (20 мин)

**Файл:** `app/services/chatterbox_service.py` (создать новый)

```python
"""
Chatterbox TTS Service - бесплатная нейронная озвучка
https://github.com/resemble-ai/chatterbox
"""

import logging
from pathlib import Path
from typing import Optional
import tempfile

logger = logging.getLogger(__name__)


class ChatterboxService:
    """
    Сервис нейронной озвучки на основе Chatterbox TTS.
    Бесплатный, локальный, с поддержкой русского языка.
    """
    
    def __init__(self):
        """Инициализация модели"""
        try:
            from chatterbox.tts import ChatterboxTTS
            
            logger.info("Loading Chatterbox TTS model...")
            self.model = ChatterboxTTS.from_pretrained("v1_0")
            logger.info("Chatterbox TTS model loaded successfully")
            
        except ImportError as e:
            logger.error(f"Failed to import Chatterbox: {e}")
            raise ImportError(
                "Chatterbox TTS not installed. Run: pip install chatterbox-tts"
            )
        except Exception as e:
            logger.error(f"Failed to load Chatterbox model: {e}")
            raise
    
    def synthesize(
        self,
        text: str,
        output_path: Optional[str] = None
    ) -> str:
        """
        Синтез речи из текста.
        
        Args:
            text: Текст для озвучки
            output_path: Путь для сохранения файла (опционально)
        
        Returns:
            Путь к WAV файлу с аудио
        """
        try:
            logger.info(f"Synthesizing TTS for text: {text[:50]}...")
            
            # Генерируем аудио (нейтральная эмоция по умолчанию)
            audio = self.model.synthesize(
                text=text,
                emotion="neutral"
            )
            
            # Сохраняем в файл
            if output_path:
                Path(output_path).parent.mkdir(parents=True, exist_ok=True)
                with open(output_path, 'wb') as f:
                    f.write(audio)
                logger.info(f"Saved audio to {output_path}")
                return output_path
            else:
                # Временный файл
                temp_file = tempfile.NamedTemporaryFile(
                    suffix='.wav',
                    delete=False
                )
                temp_file.write(audio)
                temp_file.close()
                return temp_file.name
                
        except Exception as e:
            logger.error(f"TTS synthesis failed: {e}")
            raise
    
    def get_audio_duration(self, audio_path: str) -> float:
        """
        Получить длительность аудио в секундах.
        """
        try:
            import wave
            
            with wave.open(audio_path, 'rb') as wf:
                frames = wf.getnframes()
                rate = wf.getframerate()
                return frames / float(rate)
                
        except Exception as e:
            logger.error(f"Failed to get audio duration: {e}")
            return 0.0
```

## ⚠️ Возможные проблемы

### 1. Chatterbox не устанавливается

**Симптомы:**
```
ERROR: Could not find a version that satisfies the requirement chatterbox-tts
```

**Решение:**
```bash
# Проверить Python версию
python --version  # должен быть 3.8+

# Обновить pip
pip install --upgrade pip

# Попробовать установить заново
pip install chatterbox-tts --no-cache-dir
```

### 2. Ошибка при загрузке модели

**Симптомы:**
```
Failed to load Chatterbox model: CUDA out of memory
```

**Решение:**
```bash
# Проверить память
docker stats autodoc-ai

# Добавить памяти в docker-compose.yml
autodoc-ai:
  deploy:
    resources:
      limits:
        memory: 4G
```

### 3. Генерация очень медленная

**Симптомы:**
- Один шаг генерируется >10 секунд

**Причина:** Нет GPU, работает на CPU

**Решение:** 
- Оставить как есть (просто дольше)
- Или использовать качество `standard` вместо `hd`

### 4. Ошибка импорта в API

**Симптомы:**
```
ImportError: cannot import name 'ChatterboxService'
```

**Решение:**
```bash
# Проверить что файл создан
ls app/services/chatterbox_service.py

# Проверить что класс экспортируется
grep "class ChatterboxService" app/services/chatterbox_service.py

# Перезапустить контейнер
docker compose restart autodoc-ai
```
