ИНСТРУКЦИЯ ПО ПЕРЕЗАГРУЗКЕ РАСШИРЕНИЯ
======================================

Если после обновления кода интерфейс не изменился, выполните следующие шаги:

СПОСОБ 1: Полная перезагрузка (РЕКОМЕНДУЕТСЯ)
----------------------------------------------
1. Откройте chrome://extensions/
2. Найдите "НИР-Документ"
3. Нажмите кнопку "Удалить" (Remove)
4. Нажмите "Загрузить распакованное расширение" (Load unpacked)
5. Выберите папку extension/
6. Готово! Откройте расширение заново

СПОСОБ 2: Быстрая перезагрузка
-------------------------------
1. Откройте chrome://extensions/
2. Найдите "НИР-Документ"
3. Нажмите кнопку обновления (🔄) рядом с расширением
4. Закройте popup расширения (если открыт)
5. Откройте расширение заново

СПОСОБ 3: Очистка кеша
-----------------------
1. Откройте chrome://extensions/
2. Нажмите 🔄 для обновления расширения
3. Откройте popup расширения
4. Нажмите F12 (откроется DevTools)
5. Нажмите правой кнопкой на кнопку обновления в браузере
6. Выберите "Очистить кеш и жесткая перезагрузка"
7. Закройте DevTools
8. Откройте расширение заново

ПРОВЕРКА ИЗМЕНЕНИЙ
------------------
После перезагрузки вы должны увидеть:
✓ Только одну кнопку "Начать" (до начала записи)
✓ При записи появляются кнопки: "⏸ Пауза", "⏹ Стоп"
✓ Дополнительные кнопки: ↶ (отмена клика), 🎤 (микрофон), ✕ Отменить
✓ Индикатор "⏸️ Запись на паузе" (при паузе)

ЕСЛИ НЕ ПОМОГЛО
---------------
1. Полностью закройте Chrome
2. Откройте Chrome заново
3. Перейдите в chrome://extensions/
4. Удалите расширение
5. Загрузите заново из папки extension/
