# НИР-Документ - Chrome Extension

Расширение для записи действий пользователя и создания интерактивных руководств.

## Возможности

### Основные функции
- 🎬 **Запись действий** - автоматическая запись кликов и навигации
- ⏸️ **Пауза/Возобновление** - поставьте запись на паузу, если отвлеклись
- ✕ **Отмена записи** - начните заново, если что-то пошло не так
- ↶ **Отмена последнего клика** - удалите ошибочный клик
- 🎤 **Голосовые заметки** (скоро) - добавляйте голосовые комментарии к шагам
- 📊 **Статистика в реальном времени** - видите количество кликов и время записи

### Интерфейс при записи
- Окно расширения остается открытым во время записи
- Отображается текущее время записи (без учета пауз)
- Счетчик сделанных кликов
- Индикатор паузы
- Все кнопки управления доступны

## Установка

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `extension`

## Использование

1. Нажмите на иконку расширения
2. Введите название руководства
3. Нажмите "Начать запись"
4. Выберите вкладку для записи
5. Выполняйте действия на странице

### Управление во время записи

#### Кнопки управления
- **⏸ Пауза** - приостановить запись (время паузы не учитывается в итоговой длительности)
- **▶ Продолжить** - возобновить запись после паузы
- **⏹ Стоп** - завершить и сохранить запись

#### Дополнительные функции
- **↶** - удалить последний клик (если ошиблись)
- **🎤** - голосовые заметки (функция в разработке, пока заглушка)
- **✕ Отменить** - отменить всю запись и начать заново (с подтверждением)

### Статистика
- **Статус** - текущее состояние (Готов / Запись / На паузе)
- **Время** - длительность записи без учета пауз
- **Кликов** - количество зафиксированных кликов

## Технические детали

- Manifest V3
- Работает на всех вкладках
- Автоматические скриншоты при каждом клике
- Отслеживание навигации и переходов
- Учет времени паузы в итоговой длительности
- Возможность редактировать название во время записи

## API

Расширение отправляет данные на бэкенд:
- Endpoint: `http://localhost:8888/api/v1/sessions/upload`
- Формат: multipart/form-data
- Данные: clicks.json + скриншоты

## Структура файлов

```
extension/
├── manifest.json          # Манифест расширения
├── background.js          # Service Worker (управление записью)
├── content.js             # Content Script (логирование кликов)
├── popup.html             # UI popup
├── popup.js               # Логика popup
└── icons/                 # Иконки расширения
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

## Отладка

### Просмотр логов

1. Откройте `chrome://extensions/`
2. Найдите НИР-Документ
3. Нажмите **"Inspect views: service worker"**
4. Откроется консоль с логами

### Частые проблемы

**Расширение не загружается:**
- Проверьте что все файлы на месте
- Убедитесь что иконки созданы
- Проверьте manifest.json на ошибки

**Запись не начинается:**
- Разрешите доступ к захвату экрана
- Проверьте консоль расширения

**Клики не логируются:**
- Перезагрузите страницу
- Проверьте что content.js загружен
- Откройте консоль браузера (F12)
