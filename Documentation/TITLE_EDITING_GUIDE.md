# Редактирование названия гайда

## Как изменить название гайда

1. **Откройте редактор гайда:**
   - Перейдите на http://localhost:3000
   - Выберите нужный гайд из списка
   - Нажмите на него для открытия редактора

2. **Измените название:**
   - В верхней части редактора нажмите на название гайда
   - Появится поле ввода
   - Введите новое название
   - Нажмите Enter или кликните вне поля для сохранения

3. **Отмена изменений:**
   - Нажмите Escape для отмены редактирования
   - Или очистите поле и нажмите Enter - вернется старое название

## Особенности

- ✅ **Автосохранение** - изменения сохраняются сразу при подтверждении
- ✅ **Валидация** - название должно содержать минимум 3 символа
- ✅ **Визуальная обратная связь** - поле подсвечивается при редактировании
- ✅ **Удобство** - можно редактировать прямо в интерфейсе без дополнительных форм

## Технические детали

- Изменения отправляются через PATCH запрос к API
- Обновляется поле `title` в базе данных
- Интерфейс обновляется мгновенно после успешного сохранения
- При ошибке сохранения название возвращается к предыдущему значению

## Горячие клавиши

- **Enter** - сохранить изменения
- **Escape** - отменить редактирование
- **Клик вне поля** - сохранить изменения
