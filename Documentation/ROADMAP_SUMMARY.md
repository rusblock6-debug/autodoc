1. **MediaRecorder в popup.js** (НЕ в background.js) - service workers не имеют доступа к `navigator.mediaDevices`

4. **Обработка закрытия popup** - `beforeunload` обработчик + UI предупреждение + опционально Offscreen API

## 🏗️ Архитектура (AI vs FFmpeg)

### AI делает (понимание):
- ✅ Whisper: речь → текст с таймкодами
- ✅ LLM: генерация описаний
- ✅ TTS: текст → озвучка

### FFmpeg делает (обработка):
- ✅ Нарезка video.webm по таймкодам
- ✅ Извлечение кадров
- ✅ Добавление маркеров
- ✅ Склейка сегментов
- ✅ Замена аудио

### Почему FFmpeg незаменим:
- Нарезка 10-минутного видео = 2 секунды
- Стабильно, без потери качества
- AI не умеет нарезать видео по таймкодам

## ❓ Частые вопросы

### Q: Зачем нужен FFmpeg если есть AI?
**A:** AI понимает контент, FFmpeg обрабатывает файлы. AI не может нарезать видео.

### Q: Можно ли использовать только AI?
**A:** Нет. Генерация видео из скриншотов требует A100 GPU и очень сложна.

### Q: Что если нет GPU?
**A:** Есть fallback на OpenRouter API (бесплатно) + Edge TTS.
